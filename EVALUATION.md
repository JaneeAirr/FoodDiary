## Минимальный обязательный функционал

### Frontend (React) ✅

- ✅ **Главная страница**: Dashboard с обзором питания
- ✅ **Адаптивный интерфейс**: Material-UI обеспечивает адаптивность
- ✅ **Страницы навигации**: 7 страниц с React Router
- ✅ **Формы (ввод данных)**: 
  - Регистрация/Вход
  - Добавление приема пищи
  - Установка целей
- ✅ **Таблицы / карточки**: 
  - Карточки продуктов
  - Карточки приемов пищи
  - Таблицы статистики
- ✅ **React компоненты**: 
  - Компонентная структура (components/, pages/)
  - Переиспользуемые компоненты (Navbar, PrivateRoute)
- ✅ **Маршрутизация (React Router)**: 
  - BrowserRouter
  - Routes и Route
  - Navigate для редиректов
- ✅ **Стейт (useState/useEffect)**: 
  - Используется во всех страницах
  - Context API для глобального состояния (AuthContext)
  - useCallback для оптимизации
- ✅ **Работа с API (axios)**: 
  - Сервис api.js для всех запросов
  - Interceptors для токенов
  - Обработка ошибок

### Backend (Django) ✅

- ✅ **Маршрутизация (эндпоинты)**: 
  - Django REST Framework
  - ViewSets и API views
  - 15+ endpoints
- ✅ **Работа с базой данных (SQLite)**: 
  - Django ORM
  - Миграции
  - 4 модели данных
- ✅ **CRUD-функционал**: 
  - Create: POST /api/meals/, /api/goals/
  - Read: GET /api/meals/, /api/foods/, /api/daily-summary/
  - Update: PUT /api/meals/{id}/, /api/goals/my_goals/
  - Delete: DELETE /api/meals/{id}/
- ✅ **Авторизация/регистрация**: 
  - JWT токены (djangorestframework-simplejwt)
  - Регистрация: POST /api/auth/register/
  - Вход: POST /api/auth/login/
  - Защищенные маршруты
- ✅ **Валидация данных**: 
  - Django serializers
  - Валидация на уровне модели
  - Валидация паролей
- ✅ **Возврат JSON для React**: 
  - Все endpoints возвращают JSON
  - DRF Response
  - Правильные HTTP статусы

### Общий стек ✅

- ✅ **Клиентская часть (UI)**: React + Material-UI
- ✅ **Серверная часть (логика)**: Django REST Framework
- ✅ **База данных**: SQLite (разработка)
- ✅ **Взаимодействие между слоями**: 
  - REST API
  - CORS настроен
  - JWT аутентификация
